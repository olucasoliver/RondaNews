# Sistema de Ronda - Apuração Jornalística

Sistema web para gestão de rondas de apuração e monitoramento de informações para redações jornalísticas.

## Funcionalidades

- ✅ Dashboard com quadro Kanban (Em Andamento, Pista Quente, Concluída)
- ✅ Registro de novas apurações
- ✅ Mural da Redação em tempo real
- ✅ Movimentação de cards entre colunas
- ✅ Armazenamento local (localStorage)
- ✅ Interface responsiva e moderna

## Como Usar

### Opção 1: GitHub Pages (Recomendado)

1. Faça upload destes 4 arquivos para um repositório no GitHub
2. Vá em Settings > Pages
3. Em "Source", selecione "Deploy from a branch"
4. Selecione a branch "main" e pasta "/ (root)"
5. Clique em Save
6. Aguarde alguns minutos e acesse o link fornecido

### Opção 2: Vercel (Mais rápido)

1. Crie uma conta em https://vercel.com
2. Clique em "Add New Project"
3. Importe seu repositório do GitHub
4. Clique em "Deploy"
5. Pronto! Seu sistema estará no ar

### Opção 3: Teste Local

1. Baixe todos os arquivos para uma pasta
2. Abra o arquivo `index.html` no seu navegador
3. O sistema funcionará localmente
